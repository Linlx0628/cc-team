为什么选 CC-TEAM

◈

轻巧

单文件 Node.js，零外部依赖。Docker 镜像不到 50MB，内存占用约 15MB。不需要数据库，不需要 Redis，文件存储就够了。

⬡

自由

多上游热切换，虚拟密钥映射，模型别名。想接哪个 AI 厂商都可以，配置文件改了立刻生效，不用重启。

⬢

全能

透明代理零侵入、按人用量监控、可视化仪表盘、配额管理自动调节、熔断保护、请求分类统计。一个文件全搞定。

功能一览

代理

透明代理

请求原样转发，响应原样返回。对 LLM 的上下文质量和输出零影响。

密钥

虚拟密钥

每人分配 jx-* 虚拟密钥，真实 API Key 不暴露。支持启用/禁用/过期时间。

监控

用量仪表盘

按人、按模型、按天的 Token 消耗图表。个人用量页面，团队成员自查。

配额

配额管理

按 Profile + 用户设置每日 Token 上限。高强度用户自动增长额度。

模型

模型别名

jx-sonnet → 真实模型名。换模型不改客户端配置，一行搞定。

防护

熔断保护

上游故障自动熔断，恢复后自动放行。限速保护，防止异常暴涨。

快速部署

Docker 推荐
docker pull linlx/cc-team:latest docker run -d \ -p 6789:6789 \ -v ./config.json:/app/config.json \ -v ./data.json:/app/data.json \ --name cc-team \ linlx/cc-team:latest

或者用 docker compose：docker compose up -d

脚本启动 无需 Docker
# 克隆仓库 git clone https://github.com/Linlx0628/cc-team.git cd cc-team # macOS / Linux ./start.sh # Windows start.bat

需要 Node.js >= 18，首次运行自动创建配置文件

接入方式

# 设置环境变量即可接入 export ANTHROPIC_BASE_URL="http://localhost:6789/v1" export ANTHROPIC_API_KEY="jx-your-virtual-key"
